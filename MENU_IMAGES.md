## How to Update Menu Images

### Option 1: Use Web URLs (Recommended)
1. Find an image online (from Unsplash, Imgur, or any image hosting service)
2. Right-click the image and select "Copy image address"
3. Go to "Manage Menu" tab
4. Edit the menu item
5. Paste the image URL in the "Image URL or Path" field
6. Click "Update Item"

### Option 2: Use Local Images
1. Place your images in the `images/` folder
2. Use the path: `images/your-image-name.jpg`
3. Update the menu item with this path

### Option 3: Upload to Image Hosting
1. Upload your images to:
   - Imgur: https://imgur.com
   - ImgBB: https://imgbb.com
   - Cloudinary: https://cloudinary.com
2. Copy the direct image URL
3. Use it in your menu items

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Recommended Size**: 400x300px or larger
- **File Size**: Keep under 2MB for faster loading
- **Aspect Ratio**: 4:3 or 16:9 works best

## Example Image URLs

You can use any of these formats:
- `https://example.com/image.jpg`
- `https://i.imgur.com/abc123.jpg`
- `images/local-image.jpg`
- `https://images.unsplash.com/photo-1234567890`

## Tips

- Use high-quality images for better presentation
- Ensure images are properly cropped and centered
- Test images load correctly after adding
- If an image fails to load, a placeholder will appear automatically
