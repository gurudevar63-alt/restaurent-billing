SOLUTION 2: CHECK FIREWALL (Most Common Issue!)
================================================================================

Windows Firewall might be blocking the connection:

1. Open Windows Security
2. Go to Firewall & network protection
3. Click "Allow an app through firewall"
4. Find Python and check "Private" network
5. OR temporarily disable firewall to test

Quick way:
- Settings > Windows Security > Firewall > Turn off (temporarily)
- Test connection
- Turn back on after testing

================================================================================
SOLUTION 3: VERIFY SAME WIFI
================================================================================

Both devices MUST be on same WiFi:

1. Check WiFi name on computer
2. Check WiFi name on mobile
3. They must match exactly!
4. Try disconnecting and reconnecting mobile WiFi
